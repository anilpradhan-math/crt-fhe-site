# CRT-FHE Site

This folder is a standalone GitHub Pages site source for CRT-FHE.

## What to do with it

1. Create a new GitHub repository for the project site.
2. Copy the contents of this folder into the root of that repository.
3. Push the repository to GitHub.
4. In the repository settings, enable GitHub Pages from the `main` branch root.

## Suggested repo name

- `crt-fhe`
- `crt-fhe-project`
- `crt-fhe-site`

## If you want the site to link to code

Add a source repository link in the site header or README, for example:

- `https://github.com/anilpradhan-math/crt-fhe-rlwe`

## Local preview

If you use Jekyll locally, serve the repo root and open the generated site in a browser.
